Introducción:

La creación de esta pagina web surge de la combinación del repaso para un examen de la asignatura de Aplicaciones web, y del propósito personal de desarrollar una pagina web en la que poder dar a conocer los recursos que voy recopilando y proyectos realizador por mi para ayudar a otras personas en su aprendizaje.

Si bien esta era mi primera página web, para este proyecto me propuse varias premisas, una semana de tiempo máximo para finalizarla, utilizar únicamente HTML y CSS y desarrollarla de forma individual, pudiendo buscar recursos en internet como W3C School y youtube.

Diseño de la pagina

Antes de empezar con el código primero debía pensar el contenido que tendría y como seria la página web, para la parte del contenido lo único que debía hacer fue recopilar todos los recursos que me han sido de utilidad, para hacer el diseño empecé buscando referencias, una vez tenía decidido como seria conceptualmente, utilice Miro para hacer el prototipo del diseño, Miro es una plataforma de diseño visual intuitiva, la encontré durante el proceso y la recomiendo bastante.

Desarrollo de la pagina

Una vez tenia el contenido de la pagina y tenia un diseño preliminar de como seria, ya podía empezar con el código, utilice Visual studio Code como editor de código, durante el proceso encontré algunos canales de youtube, videos y listas de Youtube que me fueron de utilizad, los cuales dejare en la sección Conclusiones.

Conclusiones

De este proyecto saco varias conclusiones, si bien el proyecto tenía fecha límite una semana, tratare de mejorarla poco a poco, hacerla mas responsive e irla actualizando, además de quizá llevarla a cabo en un CMS

Durante el proceso trate de utilizar la herramienta de Open AI, Chat GPT, si bien es la primera vez que la utilizaba mi experiencia ha sido que es una herramienta útil para funciones muy concretas como pedirle que explique algunas etiquetas y estilos o ver por donde puedes tirar para lo que quieres conseguir, sim embargo para funciones mas generales como pedirle que genere partes del código es contraproducente ya que este no se adapta y suele estar roto, mi recomendación para esto seria ver como soluciona la función que le solicites, aprender de su solución y adaptarla a tu código

como en cualquier proyecto he encontrado algunos recursos que me han sido de utilidad, algunos de estos han sido W3C School, los canales de youtube de MoureDev, Midudev y Falcon masters, La pagina web Miro para el diseño
